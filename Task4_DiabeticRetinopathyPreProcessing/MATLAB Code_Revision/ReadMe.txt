=== READ ME ==

Untuk menjalankan program:
1. set path source image dan source code pada MATLAB,
2. Jalankan "Main.m",
3. Program akan otomatis berjalan, dan menunjukkan progres dari proses yang dilakukan,
4. Tunggu hingga program selesai berjalan.

== Tambahan ==
- untuk merubah jumlah gambar yang ingin diproses, 
  -> buka "Main.m"
  -> pada line 16: rubah "n = MaxList;" menjadi misalkan "n = 100;"

- Untuk menampilkan gambar dan perbandingan normalisasi metode 1, 2 dan 3
  -> buka "doPreprocessing.m"
  -> pada line 29: rubah "TESTMODE = 0;" menjadi "TESTMODE = 1;"
